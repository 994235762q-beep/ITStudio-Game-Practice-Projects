# ITStudio Game Practice Projects

这是我在大学期间加入 ITStudio 社团游戏开发部门时练习制作的项目合集。这些项目见证了我在游戏开发领域的学习与成长历程，从最初的 Unity 小游戏到 RPG 项目开发，涵盖了多种游戏类型和开发技术。

所有项目均从旧 GitHub 账号 [@Taubor](https://github.com/Taubor) 迁移而来。

---

## 项目列表

### 1. ITstudio_game_demo

- **上传时间**: 2018 年 10 月 20 日
- **原仓库**: [Taubor/ITstudio_game_demo](https://github.com/Taubor/ITstudio_game_demo)
- **描述**: ITStudio 社团入部考核项目合集，包含三个小游戏 Demo：
  - **Angry Bird** — 愤怒的小鸟仿制版
  - **Flappy Bird** — 飞扬的小鸟仿制版
  - **My TankWar** — 坦克大战仿制版
- **技术栈**: Unity, C#
- **关键提交**: 首次提交于 2018-10-19，最后一次提交于 2018-10-27

### 2. AngryBird

- **上传时间**: 2018 年 12 月 6 日
- **原仓库**: [Taubor/AngryBird](https://github.com/Taubor/AngryBird)
- **描述**: 使用 Unity 引擎制作的愤怒的小鸟游戏。玩家通过拖拽弹弓控制小鸟的弹射角度和力度，击中并消灭所有绿色猪头即可过关。包含多个关卡，支持分数累计和特殊技能（加速、分裂）。
- **技术栈**: Unity, C#
- **关键提交**: 首次提交于 2018-12-06，最后一次提交于 2018-12-13

### 3. MySocket

- **上传时间**: 2019 年 3 月 5 日
- **原仓库**: [Taubor/MySocket](https://github.com/Taubor/MySocket)
- **描述**: 基于 C# 的 Socket 网络通信测试工具，用于在 Visual Studio 中学习和测试 Git 版本控制以及网络编程基础。
- **技术栈**: C#, .NET, Socket 编程
- **关键提交**: 提交于 2019-03-05

### 4. RPG-Project

- **上传时间**: 2019 年 3 月 11 日
- **原仓库**: [Taubor/RPG-Project](https://github.com/Taubor/RPG-Project)
- **描述**: 大一下学期（第二学期）的第一个项目。一个典型的 RPG 角色扮演游戏，包含三个场景，具备完整的游戏循环。是一个真正"活起来"的游戏项目。
- **技术栈**: Unity, C#
- **关键提交**: 首次提交于 2019-03-11，最后一次提交于 2019-03-13
